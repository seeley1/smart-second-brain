# 示例Wiki层笔记 - 如何搭建AI第二大脑

## 📝 精炼知识

AI第二大脑是一个使用AI技术自动化知识管理的系统。通过WorkBuddy + Obsidian，可以实现：

1. **多源输入** - 自动采集抖音、公众号、微信读书等知识
2. **自动消化** - AI自动解析、总结、同步笔记
3. **一键输出** - 快速生成文章、视频、社交媒体内容

## 🎯 核心要点

### 1. 系统架构

```
多源输入 → AI自动消化 → 三层知识结构 → 一键输出
```

### 2. 三层知识结构

- **00-Raw（原始层）** - 存放原始材料
- **10-Schema（结构化层）** - 存放结构化知识
- **20-Wiki（精炼层）** - 存放可输出的精炼知识

### 3. 自动化任务

- 每日知识同步（10:00）
- Obsidian对话同步（每小时）
- 知识质量验证（22:30）
- 知识精炼周度任务（每周日20:00）

## 💡 实践指导

### 搭建步骤

1. **安装Obsidian** - 创建知识库
2. **安装WorkBuddy** - AI助手平台
3. **导入模板** - 三层笔记模板
4. **配置自动化** - 4个自动化任务
5. **开始使用** - 采集、消化、输出

### 最佳实践

- ✅ 每天坚持采集知识（至少1条）
- ✅ 每周回顾和精炼知识
- ✅ 定期输出内容（文章/视频）
- ✅ 保持知识库整洁（定期归档）

### 常见误区

- ❌ 只采集不消化 → 变成"收藏家"
- ❌ 只消化不输出 → 知识无法复利
- ❌ 没有规范 → 知识库混乱
