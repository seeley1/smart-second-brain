## 质量控制标准

### 知识质量验证

#### 1. 跨源验证
- **标准**：知识是否来自多个可靠来源
- **阈值**：至少2个独立来源
- **不通过处理**：标记为"需补充来源"

#### 2. 生成力验证
- **标准**：知识是否能生成行动建议
- **阈值**：能生成具体行动建议
- **不通过处理**：标记为"需补充实践指导"

#### 3. 排他性验证
- **标准**：知识是否有独特价值
- **阈值**：有明显独特价值
- **不通过处理**：标记为"需强化独特价值"

### 质量评分

- ⭐⭐⭐⭐⭐ (5分) - 高质量，可直接输出
- ⭐⭐⭐⭐ (4分) - 较高质量，稍作修改可输出
- ⭐⭐⭐ (3分) - 中等质量，需要补充内容
- ⭐⭐ (2分) - 较低质量，需要大幅修改
- ⭐ (1分) - 低质量，建议不采纳

## 自动化任务配置

### 1. 知识库每日同步

**执行时间**：每天10:00

**功能**：
- 自动同步抖音、公众号、微信读书、ima知识库
- 自动解析内容，生成摘要
- 自动添加到Raw层

### 2. Obsidian对话同步

**执行时间**：每小时

**功能**：
- 自动同步WorkBuddy对话内容到Obsidian
- 根据关键词自动分类
- 自动添加到Schema层

### 3. 知识质量验证

**执行时间**：每天22:30

**功能**：
- 对今日新增知识进行质量验证
- 生成质量验证报告
- 标记低质量知识

### 4. 知识精炼周度任务

**执行时间**：每周日20:00

**功能**：
- 对本周知识进行精炼和优化
- 合并相关笔记
- 更新知识图谱
- 生成周度总结

## 知识复利工程

### 复利效应

1. **知识连接**：知识之间的连接越多，价值越大
2. **知识精炼**：知识越精炼，越容易产生新想法
3. **知识输出**：知识输出越多，输入越有方向

### 复利工程实施

1. **定期回顾**：每周回顾本周知识，建立新连接
2. **定期精炼**：每周精炼本周知识，提升质量
3. **定期输出**：每周输出本周知识，形成产品

## 故障排查指南

### 常见问题

#### 1. 自动化任务不执行

**可能原因**：
- WorkBuddy未运行
- 自动化配置错误
- API Key失效

**解决方法**：
- 检查WorkBuddy是否运行
- 检查自动化配置
- 检查API Key是否有效

#### 2. 笔记格式错误

**可能原因**：
- 模板文件损坏
- AI解析错误

**解决方法**：
- 重新复制模板文件
- 手动修正笔记格式

#### 3. 知识图谱不生成

**可能原因**：
- D3.js加载失败
- 数据格式错误

**解决方法**：
- 检查网络连接
- 检查数据格式
